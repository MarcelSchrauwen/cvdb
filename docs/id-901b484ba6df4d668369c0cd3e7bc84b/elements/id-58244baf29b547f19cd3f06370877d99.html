Authenticatie voorziening API ()
Via de authenticatievoorziening API kan verbonden worden met de authenticatievoorziening en daardoor gebruik gemaakt van deauthenticatieservice van die voorziening. Authenticatie is de verificatie van een beweerde identiteit. Met authenticatiemiddelen kan een persoon aantonen wie hij is en bewijzen dat hij daadwerkelijk die identiteit bezit. Het doel is om gewaarborgd vertrouwen te creëren bij de andere partij. Na elektronische identificatie volgt de bevestiging van de echtheid van een opgegeven of kenbaar gemaakte identiteit. Deze bevestiging wordt meestal verstrekt door een derde partij die de identiteit heeft gecontroleerd en vastgelegd. Authenticatie maakt dit bevestigingsproces mogelijk.
