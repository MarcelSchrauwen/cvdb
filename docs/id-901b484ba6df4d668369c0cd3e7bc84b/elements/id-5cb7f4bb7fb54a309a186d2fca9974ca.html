Identificatie voorziening API ()
Via de Identificatievoorziening API kan verbonden worden met de identificatievoorziening en daardoor gebruik gemaakt van de identificatieservice van die voorziening. Identificatie is het toekennen van kenmerken waarmee de identiteit van een persoon, organisatie of component eenduidig kan worden vastgesteld. Om zorgverleners, zorgaanbieders en patiënten binnen en tussen instellingen te kunnen herkennen, moeten deze actoren uniek identificeerbaar zijn. Bij de uitwisseling van gegevens tussen zorgverleners en patiënten speelt identificatie een cruciale rol, omdat dit de basis vormt voor verdere controles.
