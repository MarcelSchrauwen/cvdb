3. Data-integratie ()
Definitie: Gebied waarin generieke integratiefuncties zijn gepositioneerd. Deze functionaliteit maakt het mogelijk om gezondheidsdata uit de data-ontsluitingsgebied en/of uit de data-conformiteitsgebied samen te brengen. De integratiefuncties bestaan uit de functie verzamelen (verzamelen data uit de bronnen) en de functie reconciliëren (samen brengen tot één samenhangend geheel).Hierbij wordt de gezondheidsdata aangeleverd en uitgeleverd conform het uniforme (inter)nationaal informatiemodel en uitwisselingsstandaarden. Nadere duiding: - Het samenhangend geheel (dataset) wordt door de toepassing omgezet naar de gewenste functionaliteit. Bijvoorbeeld een chronologisch, eenduidig en correct overzicht voor het (virtuele) integraal gezondheidsoverzicht. De scope van het integratiegebied is specifiek voor integratie van gezondheidsdata. - In het data-integratiegebied wordt geen data bewaard - De functies in het data-integratiegebied kan bewerker zijn bij het samenbrengen tot één samenhangend geheel (ook zonder de brondata te manipuleren) - Kwaliteitscontrole kan op meerdere lagen plaatsvinden, niet alleen in het data-integratiegebied, maar plausibiliteit-controles zijn onderdeel van het verwerken van data voordat deze wordt uitgegeven aan toepassingen die data afnemen.
