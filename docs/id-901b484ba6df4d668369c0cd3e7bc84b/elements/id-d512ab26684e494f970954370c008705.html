Vertrouwensafspraken ()
Vertrouwensafspraken zijn afspraken tussen partijen om de veiligheid, betrouwbaarheid en bescherming van informatie en systemen te waarborgen. Ze leggen vast hoe organisaties omgaan met vertrouwelijkheid, integriteit en beschikbaarheid van gegevens.

bestaan uit Vertrouwensafspraken Semantische afspraken
bestaan uit Vertrouwensafspraken Technische afspraken
